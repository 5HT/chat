CHAT

SYNOPSIS

CHAT is an QoS=1 one-directional (client to server) example of messaging system built on top of:
  • SYN for publish subscribe QoS message queue;
  • N2O for protocols;
  • KVX for data storage;
  • and COWBOY for Web Server.
Acknoledgments are being delivered with QoS=0 with transactional sync replication, that delivers diff updates during feed opening. It also contains simple textual WebSocket protocol for debuggin purposes. You can freely use this example with ISC license.
